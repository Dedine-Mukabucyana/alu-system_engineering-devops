# Web Stack Debugging #2

This project contains scripts for debugging web stack issues, specifically:
- Running commands as different users
- Configuring Nginx to run as a non-root user

## Tasks

### 0. Run software as another user
Script that runs the whoami command as a specified user.

### 1. Run Nginx as Nginx
Script that configures Nginx to run as the nginx user on port 8080.
